## Model Properties

Each model includes the following properties:

- **id**: Unique identifier for the model
- **model_id**: Model identifier used in API calls
- **model_name**: Human-readable model name
- **cost**: Cost per character for text-to-speech generation
- **description**: Detailed description of the model's capabilities
- **created_at**: Timestamp when the model was created
- **updated_at**: Timestamp when the model was last updated

## Using Models

Models are used in the following endpoints:

- **Text-to-Speech Generation**: `POST /text-to-speech/:model_id`
- **WebSocket Streaming**: `WS /text-to-speech` (with `model_id` in initConnection)

<Note>
  **Model Selection**: Choose the appropriate model based on your use case, latency requirements, and cost considerations. Check the model description for specific capabilities and supported dialects.
</Note>
